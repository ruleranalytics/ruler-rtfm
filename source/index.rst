Ruler Analytics Client Documentation
===========================================

Assuming that you have already signed up and have a Ruler Analytics account ...

1. Add the javascript snippet to your site
---------------------------------------------------

.. raw:: html

    <script src="https://gist.github.com/markhigham/63461a4f36d81281459eafea3d6d93da.js"></script>

2. Conversion Tracking
----------------------

- You can automatically track all form submissions on your site.

- For standard (old school) HTML forms add a conversion javascript tag to your ``checkout`` or ``thank you`` page

.. raw:: html

    <script src="https://gist.github.com/markhigham/f0b54d12dffd364a1352c647f7a16be5.js"></script>
